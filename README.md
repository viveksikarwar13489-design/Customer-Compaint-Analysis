# Customer-Compaint-Analysis
Customer Complain Analysis by Excel Data Analysis

Consumer Complaint Analysis is a structured review of customer grievances across different financial products and services. It examines complaint volumes, average resolution times, and timely response rates to identify patterns and areas needing improvement. By applying priority scoring (based on severity, frequency, and responsiveness), the analysis highlights which complaints should be addressed first when resources are limited. Visual tools like heatmaps and bubble charts make it easier to spot high‑priority issues, ensuring management can focus on the categories that most impact customer satisfaction and regulatory compliance
